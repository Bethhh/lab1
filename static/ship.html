BattleShip Survival

Description

This is the final project of a Computer Graphic class. My partner was Yalin. In order to survive the game, the user has to press the arrow keys to make the spaceship avoid those bullets coming from one direction but in different speeds. You can see the shadows of all the objects.

Technical Features

  1. Shadow Mapping
  2. Collision Detection
